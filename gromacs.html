gromacs

Freez atoms during simulation
Just create group in index.ndx file and add 'freezegrps = group_name' and 'freezedim = Y Y Y' in mdp file --> http://binf.gmu.edu/vaisman/gromacs/online/mdp_opt.html
To extract only one fram from tranjectory
gmx trjconv -f complex_md.xtc -s md_0_1.tpr -n index.ndx -b 50000 -e 50000 ---> 50000 is time in ps
gmx tools detail
http://manpages.ubuntu.com/manpages/xenial/man1/gmx.1.html
Extract the specific frame
gmx trjcov -s file.tpr -f file.xtc -b 1000 -e 1000 ---> (1000 is the time in ps)
Select backbone of group
> 19 | 'backb' ---> where 19 is group and replace single comma with double comma
Calculate RMSF of protein
'gmx rmsf -s protein.tpr -f protein_fit.xtc -o rmsf.xvg -oq rmsf.pdb -res' choose 1 for protein.
Extend MD simulation
Add no of Steps extend in new mdp file; then run command 'grompp -f new.mdp -c old.tpr -o new.tpr -t old.cpt'; then 'mdrun -s new.tpr'
combine-xtc-files
gmx trjcat -f md_0_1.xtc md_0_1_rerun.xtc md_0_1_rerun_1.xtc -n index.ndx -o combined.xtc
Remove-pbc-box
gmx trjconv -s md_0_1.tpr -f md_0_1.xtc -o complex_md.xtc -pbc nojump<<END0
0
END0
gmx trjconv -f complex_md.xtc -s md_0_1.tpr -ur compact -pbc mol -o complex_md_noPBC.xtc<<END1
0
END1
RMSD
gmx rms -s md_0_1.tpr -f md_0_1.xtc -o rmsd11.xvg -tu ns
Hygrogen_bond
gmx hbond -f md_0_1.xtc -s md_0_1.tpr -num hydrogen-bonds-intra-peptide.xvg -tu ns
Radius-of-gyration
gmx gyrate -s md_0_1.tpr -f md_0_1.xtc -o gyrate.xvg
Define GPU and threads in gmx Command
gmx -nt 6 -gpu_id 0
-gpu_id0 --> for first GPU node
1 --> for second GPU node
01 --> for first and second GPU node
