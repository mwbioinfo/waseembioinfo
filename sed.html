sed

delete line containing string
sed '/patter/d' file_name
print 5th line
sed -n 5p filename
Replace any character at 21th position of every line in a file
sed -E 's/^(.{21})./\1H/'
Replace any character from 18th to 20th position of every colum with MOL
sed -i -r 's/^(.{17})(.{3})/\1MOL/' ligand.pdb
Lower case to Upper case
sed -e 's/\(.*\)/\U\1/' filename
Insert word at the beginning of first line
sed '1s/.*/select &/g'
Delete new line characters
sed -i ':a;N;$!ba;s/\n//g'\n
Delete empty lines in files
sed '/^$/d'
Delete first 12 line
sed '1,12d'
