xmgrace

Add legend in graph
xmgrace -legend load b-amyloid -hdevice PNG -hardcopy -printfile b-amyloid_ROG.png --> use legend load
Plot certain columns
xmgrace -block file.xvg -bxy 1:3 (for 1st and 3rd colum and use -nvx file.xbg for all columns)
Plot only one data
xmgrace input.xvg -hdevice PNG -hardcopy -printfile output.png (don't put -nxy in command)
Export image by command line
xmgrace -nxy input.xvg -hdevice PNG -hardcopy -printfile output.png
